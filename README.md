# glue-sdk
Work-in-progress Bitcoin (and some more) SDK for Linux, macOS and iOS

for Linux and macOS
```sh
$ source environment
$ make all

# for debugging
$ PRINT_DEBUG=y make all
```

for iOS
```sh
$ source environment.ios64
$ make all

# for debugging
$ PRINT_DEBUG=y make all
```

for iOS Simulator
```sh
$ source environment.iossim
$ make all

# for debugging
$ PRINT_DEBUG=y make all
```
